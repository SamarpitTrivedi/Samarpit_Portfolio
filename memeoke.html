Spotify Playlist Generator

CONCEPT

The Automatic Spotify Playlist Creator is a Python-based tool that generates a Spotify playlist featuring the Top 100 Billboard songs for any user-specified date. It automates the retrieval of Billboard chart data and compiles the corresponding tracks into a new Spotify playlist, enabling users to easily explore and enjoy popular music from specific moments in time.

PROTOTYPE

The initial prototype involved manually collecting Billboard Top 100 data for a chosen date and then searching for these songs on Spotify. This process was automated by integrating web scraping or API calls to fetch the Billboard chart for the input date. The prototype evolved to authenticate with the Spotify API, create playlists dynamically, and add matched tracks, thereby streamlining the entire workflow from data acquisition to playlist creation.

FEATURES

To implement the meme-karaoke mash-up, the following shows the breakdown of features that were initially proposed and what were eventually implemented in the application:

  • Date Input for Playlist Generation : The script accepts user input in the format YYYY-MM-DD. This date is passed to the Billboard chart source URL to retrieve that dayâ€™s list of top 100 songs. The input is validated to ensure it conforms to the required format and is not a future date.
  • Billboard Top 100 Scraper : Uses the requests and BeautifulSoup libraries to scrape the Billboard Hot 100 chart from the official Billboard website. It extracts song titles and corresponding artists by parsing HTML elements, which are often enclosed in specific CSS selectors like li.o-chart-results-list__item or h3.c-title.
  • Playlist Creation on Spotify : Once the Spotify user is authenticated, the script creates a new playlist in the user's account using sp.user_playlist_create() from Spotipy. The playlist name includes the chart date for clarity, e.g., Billboard Top 100 - 2002-05-06. A custom playlist description is also added..
  • Adding Songs to Playlist The Spotify track URIs collected from the search process are added to the playlist in batches (maximum 100 per API call). The script ensures: No duplicates are added. Errors (e.g., rate limits) are handled with retries or fallbacks. Tracks are added in Billboard ranking order for historical authenticity. .

Technical Details

  • Libraries:
    • spotipy, requests, beautifulsoup4, datetime, logging, dotenv
  • Back-End: Python Flask, SQLite
  • APIs: Spotify Web API, Billboard (via web scraping)
